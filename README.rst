============================================================
Documentación del sistema de respaldo de Tipos de Cambio
============================================================

Esta aplicación permite guardar el tipo de cambio de:

* Compra
* Venta

Para usarlo puede ejecutar el siguiente código:

.. code-block:: python

    import tc as tipo_cambio
    tipo_cambio
